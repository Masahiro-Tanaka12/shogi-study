# CLAUDE.md

## Project Overview

将棋の棋譜を分析するデスクトップアプリ。

目的は定跡データベースを作ることではなく、
自分で集めた棋譜を研究しやすくすること。

このアプリはAI最善手を教えるアプリではない。

「この局面で人間がどんな手を指したか」
を調べることを重視する。

---

## Tech Stack

* Electron
* React
* TypeScript
* SQLite

---

## Current Phase

現在はMVP開発段階。

まずは以下だけ実装する。

1. KIFファイル読み込み
2. 棋譜一覧表示
3. タグ付け
4. SFENによる局面集計

---

## Development Rules

大規模な設計変更を行う前に提案すること。

新しいライブラリを追加する場合は理由を説明すること。

実装はシンプルさを優先すること。

過度な抽象化は避けること。

まず動くものを作ること。

---

## Product Philosophy

ユーザーは将棋研究者である。

自分の棋譜や収集した棋譜を分類し、
局面ごとの傾向を調べられることを重視する。

統計は評価値より優先される。

MVPでは「便利な棋譜管理ツール」を目指す。
